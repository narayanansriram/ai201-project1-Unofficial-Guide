[June 2016] This class is about 90% memorization of facts/definitions, and problem structures. The midterm and final are all basically problems you should have seen before rephrased to ensure your comprehension. If you watch the lectures, do the labs, and take the practice quizzes you'll be ready for the tests. There are 2 programs to write using socket libraries. The first requires C or Python for the client, and C++ or Java for the server. The second requires C or C++ for the server and Java or Python for the client. So if you aren't familiar with the C socket library review it, and pick Java or Python to learn as well.
